# Steel Blog 主动更新计划

## 网站结构
- index.html - 首页（原理介绍 + 最新文章）
- news.html - 新闻动态（行业资讯）
- glossary.html - 术语表（钢结构英文术语）
- listening.html - 听力中心
- english.html - 英语学习
- search.html - 搜索

## 主动更新任务

### 1. 新闻更新（每周）
- 目标：每周添加5-10条新新闻
- 来源：行业网站、公众号 RSS
- 分类：体育场馆、超高层、桥梁、技术创新、政策法规

### 2. 术语更新（每周）
- 目标：每周添加5-10个新术语
- 分类：结构、材料、连接、设计、施工
- 格式：英文术语 + 中文解释 + 图解

### 3. 首页图解更新（每周）
- 添加新的钢结构图解
- 优化现有图解展示

### 4. 内容优化
- 检查链接完整性
- 优化图片显示
- 完善响应式布局

## 更新流程
1. 收集信息（搜索、阅读行业资讯）
2. 整理内容（新闻/术语）
3. 更新HTML文件
4. 提交GitHub

## 工具
- web_search: 搜索行业新闻
- web_fetch: 抓取网页内容
- read/write: 编辑文件
- exec + git: 版本控制

## 执行频率
- 每日：检查网站运行状态
- 每周：更新新闻和术语
- 每月：全面检查和优化
