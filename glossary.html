Space Frame 网架结构

由多根杆件按照一定的网格形式通过节点连结而成的空间结构体系。常见形式包括网架、网壳等。

📝 图解：网架结构示意

→ Space Frame = 空间 + 框架

→ 由双向/三向交叉杆件组成

→ 优点：大跨度、轻量化、美观

→ 常见应用：体育馆、机场、展厅

Example 例句

"The space frame roof spans 100 meters without internal supports."

Welded Connection 焊接连接

通过加热使金属熔化并连接在一起的连接方式。是钢结构中最主要的连接形式之一。

📝 图解：焊接类型

→ Fillet Weld: 角焊缝

→ Butt Weld: 对接焊缝

→ groove: 坡口

→ Weld Metal: 焊缝金属

Example 例句

"All welds shall be inspected by ultrasonic testing."

High-Strength Steel 高强度钢

指屈服强度不低于460MPa的钢材，用于需要承受大荷载的钢结构工程。

📝 图解：钢材等级

→ Q235: fy = 235 MPa (普通)

→ Q345: fy = 345 MPa (常用)

→ Q460: fy = 460 MPa (高强)

→ fy = 屈服强度 Yield Strength

Example 例句

"High-strength steel Q460 was used for the main columns."

Moment Frame 框架体系

依靠梁柱节点的刚性连接来抵抗荷载的结构体系，能提供较大的侧向刚度。

📝 图解：框架体系

→ Moment = 力矩

→ Rigid Connection: 刚性连接

→ 抵抗弯矩 → 提供抗侧力

→ vs 铰接框架: 刚度更大

Example 例句

"The moment frame provides lateral resistance for the building."

Load Combination 荷载组合

结构设计时，考虑不同荷载同时出现的可能性而进行的荷载叠加组合。

📝 图解：基本荷载组合

→ D + L: 恒载 + 活载

→ D + L + E: 恒 + 活 + 地震

→ 1.3D + 1.5L (LRFD)

→ D=Dead, L=Live, E=Earthquake

Example 例句

"Design shall consider all possible load combinations."

Steel Erection 钢结构吊装

将工厂预制的钢结构构件运至现场，通过吊装设备安装到位的施工过程。

📝 图解：吊装流程

→ 1. 定位 Alignment

→ 2. 吊装 Lifting

→ 3. 临时固定 Temporary Fixing

→ 4. 终拧 Final Bolting

Example 例句

"Steel erection began after the foundation was completed."

Bolted Connection 螺栓连接

使用螺栓将钢结构构件连接在一起的连接方式，分为普通螺栓和高强度螺栓。

📝 图解：螺栓类型

→ High-Strength Bolt: 高强螺栓

→ Friction Type: 摩擦型

→ Bearing Type: 承压型

→ Pretension: 预拉力

Example 例句

"High-strength bolts shall be tightened to the specified torque."

Truss 桁架

由直杆组成的三角形单元构成的平面或空间结构，主要承受轴向力。

📝 图解：桁架形式

→ Warren Truss: 瓦尔伦桁架

→ Pratt Truss: 普拉特桁架

→ Howe Truss: 豪威桁架

→ 三角形: 稳定结构

Example 例句

"The roof truss spans 30 meters with steel members."

Cantilever 悬臂结构

一端固定，另一端自由的构件。主要承受弯曲应力。

📝 图解：悬臂结构

→ Cantilever = 悬臂

→ 一端固定，一端自由

→ 主要承受弯曲应力

→ 常见应用：阳台、挑檐

Deflection 挠度

结构或构件在荷载作用下产生的位移量。是正常使用极限状态的重要指标。

📝 图解：挠度概念

→ Deflection = 挠度

→ 竖向位移最大值

→ L/250, L/400 等限制

→ 影响舒适度和外观

Young's Modulus 弹性模量

材料抵抗弹性变形的能力。钢材的弹性模量约为200 GPa。

📝 图解：弹性模量

→ E = Stress / Strain

→ 钢材: E ≈ 200 GPa

→ 混凝土: E ≈ 30 GPa

→ 铝材: E ≈ 70 GPa

📝 图解：热处理
